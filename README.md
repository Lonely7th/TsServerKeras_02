TS模型的一个策略构建库(02)

    它是什么？
    -----------
    它是TS模型的一个策略构建库，这个库通过模拟人类的思维，通过对
    基础数学、统计学、逻辑、概率进行自由组合和深度学习，这个库包含
    回测系统，学习过程中被创建的策略将保存到记忆模块中，开发者和交
    易者可以随时获取到这些策略来操作。

    最新版本
    ------------------
    最新版本可以在GitHub上找到。
    https://github.com/Lonely7th/TsServerKeras_02/下的项目页面。
